# Emotion Detection

A simple real-time emotion detection project using **FER** and **OpenCV**.  
It captures video from your webcam and detects emotions on faces, displaying the most prominent emotion with a confidence score.

## Features
- Detects multiple faces in real-time
- Displays detected emotions with confidence scores
- Uses MTCNN for face detection for higher accuracy

## Requirements

```bash
pip install -r requirements.txt
```

## Usage

1. Connect a webcam to your computer.
2. Run the script:

```bash
python main.py
```

3. A window will open showing the webcam feed.
4. Press q to quit the program.
